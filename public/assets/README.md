# Assets Folder

## Required Images

Silakan tambahkan file-file berikut ke folder ini:

1. **Alfamart-Emblem.png** - Logo Alfamart
2. **Building-Logo.png** - Logo Building & Maintenance

### Rekomendasi Ukuran:

- Format: PNG dengan background transparan
- Resolusi: 200x200px atau lebih tinggi untuk kualitas optimal
- Aspect ratio: Square atau sesuai logo asli

### Cara Menambahkan:

1. Copy file logo ke folder `public/assets/`
2. Pastikan nama file sesuai dengan yang tertera di atas
3. Refresh browser jika logo tidak muncul

---

**Note:** Komponen Header akan menampilkan logo-logo ini secara otomatis setelah file ditambahkan.
